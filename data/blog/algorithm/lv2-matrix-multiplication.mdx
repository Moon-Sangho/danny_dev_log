## 문제 설명

2차원 행렬 arr1과 arr2를 입력받아, arr1에 arr2를 곱한 결과를 반환하는 함수, solution을 완성해주세요.

- 문제 링크: [프로그래머스 - 행렬의 곱셈](https://school.programmers.co.kr/learn/courses/30/lessons/12949)

### 제한 조건

- 행렬 arr1, arr2의 행과 열의 길이는 2 이상 100 이하입니다.
- 행렬 arr1, arr2의 원소는 -10 이상 20 이하인 자연수입니다.
- 곱할 수 있는 배열만 주어집니다.

## 행렬의 곱셈 개념

행렬의 곱셈은 두 행렬 A와 B가 있을 때, A의 열의 개수와 B의 행의 개수가 같아야 곱셈이 가능하다. 결과 행렬 C의 크기는 A의 행의 개수와 B의 열의 개수로 결정된다.

즉, 앞 행렬의 크기가
**m** × n 이고 뒷 행렬의 크기가
n × **p** 인 경우, 곱셈 결과로 나오는 행렬의 크기는
**m** × **p** 가 된다.

## for 문을 이용한 행렬의 곱셈 구현

```js
function solution(arr1, arr2) {
  const r1 = arr1.length; // arr1의 행 길이
  const c1 = arr1[0].length; // arr1의 열 길이
  const r2 = arr2.length; // arr2의 행 길이
  const c2 = arr2[0].length; // arr2의 열 길이

  // 결과 행렬 초기화
  const result = Array.from({ length: r1 }, () => Array(c2).fill(0));

  // arr1의 행 순회
  for (let i = 0; i < r1; i++) {
    // arr2의 열 순회
    for (let j = 0; j < c2; j++) {
      // arr1의 열 또는 arr2의 행 순회
      for (let k = 0; k < c1; k++) {
        result[i][j] += arr1[i][k] * arr2[k][j]; // 곱셈 및 누적
      }
    }
  }

  return result;
}
```

## 함수형 스타일로 행렬의 곱셈 구현

```js
function solution(arr1, arr2) {
  // 결과 행렬의 행 루프
  return arr1.map((row) =>
    // 결과 행렬의 열 루프
    arr2[0].map((_, y) =>
      // 곱/합 계산 루프
      row.reduce((acc, cur, idx) => acc + cur * arr2[idx][y], 0)
    )
  );
}
```
