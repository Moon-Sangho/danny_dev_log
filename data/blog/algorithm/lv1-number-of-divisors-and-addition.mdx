## 문제 설명

두 정수 left와 right가 매개변수로 주어집니다. left부터 right까지의 모든 수들 중에서, 약수의 개수가 짝수인 수는 더하고, 약수의 개수가 홀수인 수는 뺀 수를 return 하도록 solution 함수를 완성해주세요.

- 문제 링크: [프로그래머스 - 약수의 개수와 덧셈](https://programmers.co.kr/learn/courses/30/lessons/77884)

## 내 풀이

```javascript
function solution(l, r) {
  let result = 0

  for (let i = l; i <= r; i++) {
    let n = 0

    for (let j = 1; j <= i; j++) {
      if (i % j === 0) {
        n++
      }
    }

    if (n % 2 === 0) {
      result += i
    } else {
      result -= i
    }
  }

  return result
}

solution(13, 17) // 43
```

## 다른 사람 풀이

```javascript
function solution(left, right) {
  var answer = 0
  for (let i = left; i <= right; i++) {
    if (Number.isInteger(Math.sqrt(i))) {
      answer -= i
    } else {
      answer += i
    }
  }
  return answer
}
```

## Lesson Learned

제곱근이 정수면 약수의 개수가 홀수이다. <br />
왜 특정 수의 제곱근이 정수면 약수의 개수가 홀수가 될까?

- 약수는 두 수의 곱으로 표현되는데, 제곱근은 자기 자신과의 곱으로 표현된다.
- 예를 들어, 36의 약수는 1, 2, 3, 4, 6, 9, 12, 18, 36이다. 이 중 6은 6 \* 6 = 36로 자기 자신과의 곱으로 표현된다.

  ```
  1 × 36
  2 × 18
  3 × 12
  4 × 9
  6 × 6
  ```

- 따라서 제곱근이 정수인 경우, 약수의 개수가 홀수가 된다.
